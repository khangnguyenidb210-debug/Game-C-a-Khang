# Game-C-a-Khang
Tựa game được tạo ra bởi code C++ của Khang và Lợi và được chuyển hóa thành code html + css + javascript, cùng với sự hỗ trợ code từ gemini và ý tưởng nền móng code ban đầu đã tạo ra tựa game "Escape from the technology teacher". Lưu ý game chỉ mang tính chất giải trí và không có mục đích xúc phạm cá nhân nào khác, cảm ơn.
